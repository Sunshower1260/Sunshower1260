## 🧠 Learning Philosophy: The Axiom Sandbox

I don't learn by memorizing docs or following tutorials step-by-step. I build my engineering mindset through a self-designed methodology that blends **First-Principles Thinking**, **Socratic Dialogue**, and **Sandbox Experimentation**:

- 🔬 **Deconstruct (First Principles):** Strip every technology down to its core mechanics. Instead of just using a library, I want to understand the actual logic running beneath it.
- ❓ **Question (Socratic Method):** Constantly challenge my own code: *"Why this architecture?", "If the system scales 10x, what breaks first?"*
- 🧪 **Play (Sandbox):** Throw every hypothesis into a sandbox — build small projects, intentionally **break** them, then fix and extract real lessons.
